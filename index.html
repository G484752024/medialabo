本テーマについて

このテーマでは，HTML, CSS, JavaScript の基本を学び，これらを組み合わせて Web ページを構成する方法を学びます．学習目標としては、HTMLによって Web ページの構造を記述する基本的な方法を学んだり、HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できるといった能力を得ることが学習目標となる。

プロジェクト課題について

プロジェクト課題では、グルメ情報について検索できるWebサイトが貼られている。検索キーに「居酒屋」や「創作料理」などの具体的なお店系統(指定されているもの)を入力することで八王子付近のお店をサーチすることができる。操作方法としてはページを開いたあと検索キーの四角のマスに先ほど説明したようにお店の系統を入力し検索ボタンを押す。そうするとその系統のお店が八王子付近にあるだけ表示される。また一度検索した後に別の系統のお店を調べたくなった場合また同じ操作をすれば良い。

数当てゲーム

乱数で1~10のランダムな数の中から1つ正解の数を算出し、それに対してユーザーはその答えを予想し3回まで数を出すことができる。ユーザーはその3回の中で正解ではない数字を予想してしまった場合、その入力した数字より小さいか大きいかを教えてもらうことができる。3回内に答えを出すことができればユーザーの勝ち出すことができなければ負けとなる。

操作方法としては、1から10までの数を1つ選び、四角のマスの中に入力し解答ボタンを押す。そうするとその数がランダムに生成された答えに合っていれば正解となり終了するが、不正解の場合はもう1度予想することができる。その時、正解が選択した数より大きいか小さいかがヒントとして表示される。予想できる回数は全部で3回まで。

数当てゲームのページ:kazuate.html

工夫点，苦心点，強調点，感想

工夫点，苦心点，強調点

工夫点としては、初めから落ち着いた印象であるカフェのwebページのようなものをイメージして作った点です。色合いを茶色や白系統の色を使うことで落ち着いた印象を与えられるページにしました。しかし苦心点として,初めは大きめのコーヒーの画像を薄めにして右上あたりに配置したかったのですが、調べてみてもやり方が分からず断念してしまったところが苦心点です。強調点としては先ほども申した通り色合いを意識しました。ただ落ち着いた色合いを一番に意識したわけではなく文章が見づらくならないことを一番に意識し、作りました。特に<"h2">要素の部分は背景色をグラデーションして少しおしゃれな感じにしたのですが、グラデーションは他の色と比べて文字が見づらくなってしまったため色合いを特に気をつけて作りました。
